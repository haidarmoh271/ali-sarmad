الخطوة 2️⃣: إضافة حساب المدير في Supabase

تشغيل سكريبت SQL

  1. افتح Supabase Dashboard على الرابط: https://supabase.com/dashboard
  2. اختر مشروعك (jmwicmnwgjwkhayvpkld)
  3. من القائمة الجانبية، اختر SQL Editor
  4. انسخ والصق الكود التالي:
-- Insert Admin User for Miswaki E-commerce
-- Default admin credentials: admin@miswaki.com / Admin@123

INSERT INTO admins (full_name, email, password)
SELECT 'Admin User', 'admin@miswaki.com', 'Admin@123'
WHERE NOT EXISTS (
  SELECT 1 FROM admins WHERE email = 'admin@miswaki.com'
);
  1. اضغط على Run لتشغيل السكريبت
  2. تأكد من ظهور رسالة نجاح

الخطوة 3️⃣: تسجيل الدخول كمدير

استخدام بيانات الاعتماد الصحيحة

📧 البريد الإلكتروني: admin@miswaki.com
🔑 كلمة المرور: Admin@123

خطوات مهمة جداً:

  1. اذهب إلى صفحة تسجيل الدخول: http://localhost:5173/login
  2. ⚠️ يجب الضغط على زر "🔑 تسجيل دخول كمدير" في أعلى الصفحة لتفعيل وضع المدير
  3. أدخل البريد الإلكتروني: admin@miswaki.com
  4. أدخل كلمة المرور: Admin@123
  5. اضغط على زر "تسجيل الدخول"
⚠️ تحذير: إذا لم تضغط على زر "تسجيل دخول كمدير"، سيحاول النظام تسجيل دخولك كمستخدم عادي وليس كمدير!

الخطوة 4️⃣: التحقق من نجاح العملية

اختبار الوصول

بعد تسجيل الدخول بنجاح:

  1. يجب أن يتم توجيهك تلقائياً إلى لوحة التحكم
  2. يجب أن يظهر رابط "لوحة التحكم" في شريط التنقل
  3. يجب أن تتمكن من الوصول لجميع صفحات المدير

استكشاف الأخطاء

إذا استمرت المشكلة:

  1. تأكد من تشغيل سكريبت SQL في Supabase بنجاح
  2. امسح localStorage باستخدام الزر أعلاه
  3. أعد تحميل الصفحة بالكامل (Ctrl + Shift + R)
  4. تأكد من الضغط على زر "تسجيل دخول كمدير" قبل إدخال البيانات
  5. افتح أدوات المطور (F12) وتحقق من Console للأخطاء
